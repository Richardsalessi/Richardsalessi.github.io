📄 Explicación del Proyecto
Este es un proyecto sencillo donde se combinan HTML, CSS y JavaScript para crear una página web con contenido básico y un poco de interactividad.

HTML estructura la página con diferentes elementos.
CSS le da diseño, colores y tipografía para que se vea mejor.
JavaScript permite que el botón realice una acción cuando se le da clic.
La idea es aprender cómo estos tres lenguajes trabajan juntos en la creación de páginas web.

📚 Parte Teórica
1️⃣ ¿Qué es HTML y cuál es su función en la web?
HTML es el lenguaje usado para estructurar las páginas web. Su función es definir el contenido y los elementos que la componen, como textos, imágenes, enlaces y botones. Sin HTML, una página web no tendría orden ni sentido.

2️⃣ ¿Qué es una etiqueta HTML y menciona algunas de las más comunes?
Las etiquetas HTML son los elementos que usamos para decirle al navegador cómo mostrar el contenido. Algunas de las más usadas son <h1> para títulos, <p> para párrafos, <a> para enlaces e <img> para imágenes.

3️⃣ ¿Qué es un atributo en HTML y cuáles son los más comunes?
Los atributos son características adicionales que se pueden agregar a las etiquetas para modificar su comportamiento. Algunos de los más comunes son href (para enlaces), src (para imágenes y videos) y alt (para dar una descripción a las imágenes).

4️⃣ ¿Qué es CSS y cómo se usa en el diseño web?
CSS es el lenguaje que usamos para darle estilo a las páginas web. Nos permite cambiar colores, fuentes, tamaños de texto, márgenes y la distribución de los elementos para que la página no se vea aburrida o desordenada.

5️⃣ ¿Qué es una propiedad en CSS y menciona algunas comunes?
Las propiedades en CSS son las reglas que le damos a los elementos para cambiar su apariencia. Algunas de las más utilizadas son color (para cambiar el color del texto), background-color (para el fondo), font-size (para el tamaño de la letra) y margin (para los espacios entre elementos).

6️⃣ ¿Qué es un selector en CSS y cuáles existen?
Los selectores en CSS sirven para indicar a qué parte de la página queremos aplicar estilos. Existen selectores de etiquetas (como p para párrafos), de clases (con . antes del nombre) y de ID (con # antes del nombre). También se pueden combinar para ser más específicos.

7️⃣ ¿Qué es JavaScript y cómo añade interactividad a las páginas web?
JavaScript es el lenguaje que permite que una página web haga cosas sin necesidad de recargar. Por ejemplo, validar formularios, mostrar mensajes, cambiar imágenes al hacer clic o reaccionar a acciones del usuario.

8️⃣ ¿Cuáles son los tipos de datos primitivos en JavaScript?
Son los valores básicos que JavaScript maneja. Estos incluyen:

String: Para textos.
Number: Para números, ya sean enteros o decimales.
Boolean: Para valores true o false.
Null: Para representar la ausencia de valor.
Undefined: Cuando una variable aún no tiene un valor asignado.
BigInt: Para números muy grandes.
Symbol: Para crear identificadores únicos.
9️⃣ ¿Cómo funcionan las estructuras de control de flujo en JavaScript?
Las estructuras de control permiten que el código tome decisiones o repita acciones según ciertas condiciones.

if / else: Se usa cuando hay que decidir entre una o varias opciones.
switch: Permite evaluar múltiples casos sin escribir varios if.
Bucles (for, while, do-while): Se usan para repetir código varias veces sin tener que escribirlo manualmente.
